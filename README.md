# Campus Notification Dashboard

Next.js + TypeScript + Material UI frontend  

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Auth

On the login screen:
- **Tab 1 (Use Token):** Paste your `access_token` directly
- **Tab 2 (Authenticate):** Enter credentials to auto-authenticate

## Features

- Priority Inbox (Placement > Result > Event, sorted by recency)
- Filter by notification type
- Pagination (limit=10, page-based)
- Read/Unread visual distinction
- Responsive (mobile + desktop)
- Mark individual or all notifications as read
